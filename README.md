Title
Pencil Pops Balloon
Description
This Scratch project is a simple interactive game where the player controls a pencil to pop balloons. The objective is to pop as many balloons as possible before time runs out.

How to Play

Use arrow keys to move the pencil
Touch balloons to pop them
Score increases with each pop

Features

Score tracking

What I Learned

Through this project, I learned how to:
Use conditionals
Detect sprite collisions
Manage variables for score
